# Valuate

Stat weight calculator addon for World of Warcraft: Ascension Bronzebeard (WotLK 3.3.5a).

## Description

Valuate calculates item scores based on customizable stat weights, helping you determine which gear is best for your character and playstyle.

## Installation

1. Copy the `Valuate` folder to your `World of Warcraft/Interface/AddOns/` directory
2. Restart World of Warcraft or type `/reload` in-game
3. Enable the addon in the Character Select screen

## Current Status

**Version 0.1.0** - Initial release (basic loading only)

This is an early development version. Core functionality is being implemented incrementally.

## Usage

Type `/valuate` or `/val` in-game for available commands.

## Features (Planned)

- Tooltip integration showing item scores
- Support for Ascension-specific stats (PvE Power, PvP Power)
